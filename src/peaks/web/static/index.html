Embed

Sample & embed scenes into the cache. Skips anything already done.

Overrides apply to this run only — no restart, nothing saved to the container.


    


    

      
Failed scenes 

      
Scenes that failed to embed (usually a seek/decode quirk,
        not a bad file). Retry runs a fallback ladder: sparse without NVDEC,
        then a full linear decode like VLC uses.

      

        
        
        
        
      

      

      

    


    

      
Sync library

      
Reconcile the cache with Stash: refresh scenes that moved
        between folders and drop cache entries for scenes deleted from Stash.

      

        
        
        
      

      

    


    

      
Score

      
Find your apexes and write markers into Stash. Run once with
        write off first — the log shows a calibration read-out (your
        actual score distribution) so you can set thresholds that match.
